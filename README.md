# Omni-Loom Fabrication Compiler

**The Enterprise Deterministic Fabrication Compiler** — Every cut, compiled.

## Overview

Mid-size fabrication shops hemorrhage 780+ hours annually to format fragmentation. Omni-Loom is a universal intermediate representation (`stitch_plan.json`) that compiles creative intent to physical dispatch with mathematical determinism.

## Component System

All components are self-contained with **zero external dependencies**:

- **Particle Physics** — 60-particle canvas with mouse repulsion + fabric mesh
- **Custom Cursor** — 8-particle spring trail, mix-blend-mode:difference
- **Ambient Audio** — Web Audio API 4-oscillator harmonic drone
- **Scroll Reveal** — IntersectionObserver with directional variants
- **SVG Tactile Filters** — leather-grain, emboss-stitch, denim-texture
